# 🚀 Simple Microservice with DevOps Pipeline

A small Node.js app with full DevOps pipeline:
- Dockerized app
- CI/CD via GitHub Actions
- Docker Hub push
- Deployment on VPS
- Prometheus + Grafana monitoring

## 💻 Stack
- Node.js
- Docker, Docker Compose
- GitHub Actions
- Prometheus, Grafana
- Terraform (future)
